Lección 1: Fundamentos

¿Qué es HTML?

El lenguaje HTML es un lenguaje de programación que se puede usar principalmente para crear
sitios web públicos y privados.
También es algo fundamental por que ahí es donde se agrega el texto, contenedores,
entre muchas cosas más...

Igual el HTML además de poner texto, también puede usar etiquetas para llamar a archivos
y agregar estilo a el texto o el contenido que haya en el HTML como son los
archivos .css
Pero, por ahorita iniciemos con el tema pincipal: El lenguaje HTML.
